Notions et definitions essentielles

  • Hiérarchie : Non-maître>Capacité>Tirant>Pêche>Voilier>Moteur
  • Routes opposées : tribord-tribord
  • Routes convergentes : priorité droite
  • Doublage : s'écarter
  • Voilier+moteur : traité moteur
  • Chalutier pêche : 2 cônes, feu vert/blanc
  • Remorqueur : feu blanc supplémentaire
  • Risque : gisement constant 3 temps

Points cle a maitriser

  • Maîtriser les règles de barre (priorités de passage)
  • Identifier les feux de route des navires
  • Connaître les marques de jour
  • Appliquer la hiérarchie des privilèges entre navires

Methode de resolution en examen

  1. Identifier le type de rencontre: route opposee, croisement, rattrapage.
  2. Determiner le navire privilegie selon RIPAM.
  3. Executer une manœuvre franche, precoce et lisible.

Erreurs frequentes a eviter

  • Attendre trop longtemps avant de manœuvrer.
  • Penser qu un navire privilegie ne doit rien surveiller.
  • Confondre priorite voile/moteur avec les cas particuliers.
